# 📊 Sales Performance Dashboard (Excel)

An interactive sales analytics dashboard built in Microsoft Excel to analyse revenue, profitability, and operational performance across different business dimensions.

---

## 🚀 Project Overview

This project focuses on transforming raw sales data into meaningful business insights using Excel. The dataset was cleaned, validated, and standardised before building an interactive dashboard using PivotTables, KPI cards, and slicers.

The goal was to simulate a real-world business reporting tool that supports decision-making.

---

## 🧹 Data Preparation

The dataset was prepared using Excel functions to ensure data quality and consistency:

- Removed inconsistencies using `TRIM`
- Standardised categorical values using `XLOOKUP`
- Handled errors and missing values using `IFERROR`
- Validated business logic (Profit = Sales - Expense)

---

## 📈 Key Features

- ✔ KPI Cards for:
  - Total Sales
  - Total Profit
  - Total Orders
  - Average Order Value

- ✔ Pivot-based analysis:
  - Sales distribution by State
  - Sales by Service Category
  - Profit contribution by Sales Agent
  - Order Status breakdown

- ✔ Interactive dashboard using slicers:
  - State
  - Service Category
  - Sales Agent
  - Order Status

## 💡 Key Insights

- NSW and VIC contribute the highest share of revenue  
- Mentoring services generate the most sales  
- Certain sales agents consistently outperform others in profit contribution  
- Majority of orders are successfully delivered, with minimal cancellations  

---

## 🛠 Tools & Techniques

- Microsoft Excel  
- PivotTables & Pivot Charts  
- Slicers for interactivity  
- Data cleaning functions (TRIM, XLOOKUP, IFERROR)  
- KPI design using linked cells and shapes  

---

## ⚠️ Limitations

- Dataset contains one transaction per customer (no repeat purchase analysis possible)  
- No time-series trend analysis included  
- Limited customer-level behavioural insights
